=== Description === 

// This is the BuddyForms Members Component 
Auto Detect and integrate other custom post types to BuddyPress.  
Your users can create posts, events, products, videos, etc. right out of their member profiles.  
This way you can integrate many CPT based plugins to BuddyPress!  

// You need the BuddyForms plugin installed to work. 
Check it out at http://buddyforms.com
